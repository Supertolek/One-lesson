# One lesson

## Concept

One lesson is a windows app to store your course sheets in one place.
It's inovation is to use fluent design and the [winui kit](https://learn.microsoft.com/fr-fr/windows/apps/winui/).

## How to use it

Every versions are availlable on [Supertolek/One-lesson-download](https://github.com/Supertolek/One-lesson-download).
